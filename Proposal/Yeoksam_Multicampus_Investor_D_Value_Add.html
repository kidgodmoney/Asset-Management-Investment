  • 취득 즉시 삼성SDS와 임대 연장 협상 개시

  • 병행 과제: TDD 결과 기반 긴급 수선 항목 처리 (통상 운영비 범위 내)
  • Stage 1 완료 조건: 5년+ 임대 재계약 MOU 또는 LOI 확보
STAGE 2 리뉴얼 및 인증 Year 2~3
목표: CapEx 50억 집행, 임대료 프리미엄 기반 구축
  • Y2 (25억): 공용부 리뉴얼 20억 + LEED 인증 착수 5억
  • Y3 (25억): LEED 인증 완료 10억(잔금) + 스마트빌딩 솔루션 15억
  • CapEx 집행 기간 중 임시 불편 최소화 (공사 야간·주말 집중)
  • Y3 말 LEED Gold 인증 취득 목표 → 시장 커뮤니케이션 시작
STAGE 3 프리미엄 임대료 + Exit Year 4~5
목표: 개선된 자산가치로 고배율 Exit
  • Y4 임대 재계약 반영 임대료 현행화: 270,000 → 295,000원/평/월 (+9%) 목표
  • LEED Gold 인증 프리미엄 + 스마트빌딩 프리미엄 → 시장 최상위 임대료 포지셔닝
  • Y4~Y5: Post-CapEx NOI 155억 안정화, Exit 투자자 대상 마케팅 시작
  • Y5 Exit: NOI 155억 / Cap Rate 3.7% = 자산가치 4,189억 목표
