# ⚖️ Korean Law MCP 실무 활용 시나리오 (Asset Management)

[**🔗 Live Demo (웹페이지 바로가기)**](https://kidgodmoney.github.io/Asset-Management-Investment/mcp-scenarios/)

자산운용사 실무(투자운용, 준법감시, 리스크관리, 경영/IT 등)에 특화된 법률 인공지능(MCP) 활용 시나리오 10가지를 모아둔 쇼케이스 웹페이지입니다. 

단순한 법령 검색을 넘어 **투자검토 메모 작성, 컴플라이언스 체크리스트 자동화, 외부 계약서 리스크 점검** 등 실무에서 즉시 파급력을 낼 수 있는 구체적인 AI 프롬프트와 활용법을 제안합니다.

---

## 🎯 핵심 시나리오 요약 (Top 10 Scenarios)

### 📈 1. 투자운용 & 영업지원
* **신규 딜 구조 법적 쟁점 사전 점검** (관련: 자본시장법, 집합투자업자 의무)
* **투자제안서(IM) 초안 법률 리스크 확인** (관련: 금소법, 위험 고지 점검)
* **대외 제안서 및 브로셔 문구 사전 교정** (관련: 확정수익 오해 방지, 면책조항)

### 🛡 2. 준법감시 & 리스크관리
* **운용 프로세스별 준법 체크리스트 자동 생성** (설정-집행-사후관리 체크)
* **법령 개정 영향분석** (시행령 개정 시 내부 규정/프로세스 자동 대조)
* **분쟁 가능성이 큰 딜의 선례 조사** (불완전판매, 손실분쟁 유사 판례/해석례 서치)
* **AML / KYC 온보딩 체크리스트** (법인 LP 수탁 시 고위험 징후 탐지)

### 💼 3. 경영진 & IT/총무
* **외부업체 솔루션 계약서 리스크 1차 점검** (책임 제한, 손해배상, SLA)
* **개인정보 위탁·SaaS 보안 문서 리뷰** (처리위탁, 접근권한, 통지 의무)
* **이사회/감사 보고용 핵심 리스크 브리핑** (경영진 의사결정용 하이레벨 리포트)

## 💻 디자인 및 기술 스택 (Tech Stack)
* **Front-end**: Pure HTML5, CSS3, Vanilla JavaScript (빌드 프로세스 없는 정적 페이지)
* **UI/UX**: 
  * Premium Dark Mode & Glassmorphism (글래스모피즘) UI 적용
  * 미세한 네온 백그라운드 글로우 및 카드 오버(Hover) 애니메이션
  * 부서별 카테고리(투자, 준법, 리스크 등) 동적 필터링 기능 구현

## ✨ 관리자 가이드
이 폴더(`mcp-scenarios`) 내의 파일들은 별도의 Server나 모듈러 빌드 없이 작동합니다. 로컬 실행 시 브라우저에서 `index.html`을 바로 오픈하거나, Python 로컬 서버(`python3 -m http.server 8000`)를 띄워 확인하실 수 있습니다.
